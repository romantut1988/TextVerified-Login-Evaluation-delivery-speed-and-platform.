## 3. How Identity Verification Software Works

Identity Verification Software combines document validation and biometric verification to confirm user identity.

The process includes:

### 1. Government ID Verification

Users provide a supported identity document, such as:

* Passport
* Driver’s license
* National identity card

The system analyzes:

* Machine-readable zones
* Document security features
* Identity attributes
* Signs of possible forgery

### 2. Biometric Liveness Detection

Identity Verification Software uses biometric liveness detection to confirm that a real person is completing the verification process.

The system analyzes:

* Real-time facial movements
* Depth information
* Physical presence indicators

This helps prevent attacks using:

* Photos
* Recorded videos
* Masks
* AI-generated deepfakes

### 3. Identity Matching

Verified information is compared with existing user records to confirm that the person requesting access is the legitimate account owner.

### 4. Workflow Integration

Verification results connect directly with existing Specops workflows without requiring separate systems or complex deployment.
